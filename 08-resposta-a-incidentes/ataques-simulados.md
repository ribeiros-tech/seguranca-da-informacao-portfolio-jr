# 🚨 Simulação de Incidentes



## Tentativa de login inválido

Foi realizada uma tentativa de login com usuário inexistente, gerando registro de falha no sistema.



## Falha de autenticação

Foram realizadas tentativas com senha incorreta para usuário existente.



## Acesso indevido

Usuário comum tentou acessar diretório restrito, sendo bloqueado pelo sistema.



## Monitoramento em tempo real

Foi utilizado `tail -f` para acompanhar eventos de autenticação em tempo real.



## Análise

Os logs permitiram identificar atividades suspeitas e demonstraram a importância do monitoramento contínuo.
