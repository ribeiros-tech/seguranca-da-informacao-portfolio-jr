# 🛡️ Projeto de Monitoramento com Zabbix

## 🎯 Objetivo
Implementar um sistema de monitoramento com geração de alertas automáticos para detecção de incidentes.

## 💻 Tecnologias utilizadas
- Zabbix
- Linux (WSL)
- Windows
- SMTP (Gmail)

## ⚙️ Funcionalidades
- Monitoramento de CPU, memória e disco
- Criação de triggers para eventos críticos
- Envio de alertas por e-mail
- Simulação de incidente

## 🚨 Exemplo de alerta
Alerta gerado quando CPU ultrapassa 80%

- Configuração de monitoramento com Zabbix

Projeto demonstra habilidades em monitoramento, resposta a incidentes e automação de alertas.
